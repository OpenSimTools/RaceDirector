# RaceDirector

Currently the software is a console application that simply prints a message when RaceRoom is started and stopped (CTRL+C to stop it):

```
Starting pipeline
> RRRE64
>
```

## Build

```
dotnet test
dotnet publish
```

## Run

```
src\RaceDirector\bin\Debug\net5.0\publish\RaceDirector.exe
```
